# GIT MERGE CONFLICT

## WORKFLOW

1. Lakukan merge pada 2 branch yang ada.
2. Saat terjadi conflict, code editor akan terbuka.
3. Perbaiki line code yang menjadi conflict.
4. Tambahkan source code yang telah diperbaiki ke Staging Area.
5. Lakukan commit.
